Malaysia Payroll Management

Here you can manage the Organization easly in order to Malaysia payroll system.

Salary Rules

Salary Rule Categories

Employee Payslip : Added Pay By Cheque, Remaining Month, Total YTD Gross and Other fields in Employee payslip.
YTD (Year to Date) Gross, EPF, PCB and PCB deductions fields fetch the amount from current year's payslips.

Employee Contract

* Salary and Wages : Added Wage To Pay, Rate per hours, PCB and Socso fields in Employee contract.
* PCB Fields : Vola, BIK (Benifit in kind) and Esos are monthly additions for Employees payslips.
* Socso Fields : Employee Socso rate table is used in Socso employee and Employer's Salary rules.

Added Previous Employment boolean field, which enable the previous employment configurations page in Employee contract.

PCB Salary Rule
* Here you can manage PCB Salary Rules.

* PCB current month amount and EPFE for Employee payslip.

Payroll Reports

* Added configuration in payroll settings for Payroll Reports, PF Reports and SOCSO Report.
* Generate Payroll Reports : It will Install Malaysia Payroll Reports Module.
* Generate PF Reports : It will Install Malaysia PF Reports Module.
* Generate SOCSO Reports : It will Install Malaysia SOCSO Reports Module.
