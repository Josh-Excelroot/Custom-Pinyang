Malaysia Payroll Management

Payroll

Employee Payslip

* Added YTD (Year to Date) Gross, EPF, PCB and PCB deductions salary rules

Employee Contract: Added following fields in contract.

* Salary and Wages: Added Wage To Pay, Rate per hours, PCB and Socso fields in Employee contract.
